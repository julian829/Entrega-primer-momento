# Astrologic – Entrega Python 1

## Descripción

Astrologic es un programa desarrollado en Python que permite registrar y visualizar eventos.
El sistema funciona mediante un menú interactivo en consola donde el usuario puede guardar eventos y consultar los que ya fueron registrados.

## Funcionalidades

El programa permite:

* Registrar nuevos eventos
* Ver la lista de eventos guardados
* Salir del sistema mediante el menú

Los eventos se almacenan temporalmente en una lista utilizando diccionarios.

## Tecnologías utilizadas

* Python 3
